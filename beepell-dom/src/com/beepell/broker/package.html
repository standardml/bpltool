Provides classes and interfaces for web service invocation and publication.

Package Specification

The main class of this package is the {@link com.beepell.broker.Broker} that mediates the exchange and conversion of {@link com.beepell.broker.Message}s. On one side is the "client" that operates on abstract WSDL messages ({@link com.beepell.broker.Message}). On the other side is the {@link com.beepell.broker.SOAPProvider} that handles incoming requests and the {@link com.beepell.broker.SOAPInvokeHandler} that handles remote service invocations.

The {@link com.beepell.broker.Broker} contains queues for each WSDL port type that is published and offers direct access to examine these queues and pick out individual messages.

Note: Messages arrive in multi-threaded environment and consequently the "client" must obtain the object lock on the queue while traversing it. In general this is done as follows:

        MessageQueue queue = this.broker.getMessageQueue(portTypeName);
        Iterator iterator;
        synchronized (queue) {
            iterator = queue.getIterator();
            while(iterator.hasNext()){
                message = iterator.next();
                /* do something */
            }
        }

The {@link com.beepell.broker.Broker} and the "client" only handles abstract {@link com.beepell.broker.Message}s. The {@link com.beepell.broker.SOAPProvider} and the {@link com.beepell.broker.SOAPInvokeHandler} are responsible for converting the abstract {@link com.beepell.broker.Message} to and from {@link javax.xml.soap.SOAPMessage}.
